## Списки

Что бы добавить ненумерованные списки необходимо пункты выделить звездочкой (*).
Например, вот так:

* Элемент 1
* Элемент 2
* Элемент 3

Чтобы добавить нумерованные списки, необходимо пункты просто пронумеровать.
Например, вот так:

1. Первый элемент
2. Второй элемент
3. Третий элемент
